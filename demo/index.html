Web Component: virtual-list 

<div is="virtual-list"></div>

Styling

No styles at all in the package, here is a simple starter. (note: it's pure CSS, not SASS!) 

[is='virtual-list'] {
    overflow-y: auto;
    max-height: 200px;

    > ul {
        position: relative;
        margin: 0;
        padding: 0;

        > li {
            position: absolute;
            display: flex;
            align-items: center;
            padding-inline: 5px;
            white-space: nowrap;
            cursor: pointer;
            width: 100%;
            overflow: hidden;

            &:hover {
                background-color: #eee;
            }

            &.selected {
                background-color: #333;
                color: #fff;
            }
        }
    }
}

Basic usage

const div = document.querySelector('div')

// assign an array to `items` property to populate the list
div.items = ['item 1', 'item 2', ...]

// then listen to selection
div.addEventListener('input', () => console.log(div.items[div.selected]))

Templated items

const element = document.querySelector('div')

element.itemHeight = 35 // in px, default to 22

// define the template, it must be a function, returning a string
element.template = ({ initials, title, num }) =>
    `<b>${initials}</b><div>${title}</div><em>${num}</em>`

// and set corresponding items
element.items = [
    {
        initials: 'DE',
        title: 'gima ki dary delikuca',
        num: 27
    },
    ...
]
